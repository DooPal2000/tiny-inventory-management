# 미니 재고관리 앱

이 프로젝트는 간단한 재고관리 시스템을 구현한 웹 애플리케이션입니다.

## 기술 스택

- Node.js
- Express
- MongoDB
- EJS (Embedded JavaScript templating)
- Bootstrap
- SweetAlert
- Docker & Docker Compose
- Nginx
- HTTPS (GoGetSSL)

## 주요 기능
- 재고 항목 추가, 조회, 수정, 삭제 (CRUD)
- 사용자 인증 및 권한 관리
